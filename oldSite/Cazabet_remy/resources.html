DATA

Wikileaks


This dataset is linked to the «cablegate», the set of classified documents published by Wikileaks.

------------------------------------------------

File 1 download (.csv 3.1MB)

This file is the extraction of the headings of all the cables. It comes from the guardian. What is interesting is that for every cable, we have the writing date, and the associated keywords.

------------------------------------------------

File 2 download (.txt), browse

This file contains the meaning of (nearly) all the keywords of file1. I produced this file from a pdf document found on this blog. It’s called «U.S. Department of State Foreign Affairs Handbook Volume 5 Handbook 3 – TAGS/Terms Handbook» and it seems quite reliable.

------------------------------------------------

File 3 download (.txt)

Exemple of what we can produce by studying File1 : using ILCD (my community detection algorithm), we can detect many of the main events of the world (view from the U.S diplomacy). As an exemple, if I do a search for the keyword KFLU (avian/pandemic disease), we can see :

  1. -an event from 2006 to 2007 linked to Nigeria. (avian flu)

  2. -an event in 2009 linked to Mexico, then germany, etc..

  3. -an event from 2006 and still alive concerning medical disposition, prevention, ...

There’s many more results, let me knows if you find these dataset useful.
